Company Profile


NAC Geographic Products Inc. develops and publishes geographic
related software, CD-ROM and printed products, and provides real time geocoding, mapping, driving directions and location based business search services, and mobile location services. We have
already had software products: http://mlbs.net/, http://www.travelgis.com/, WEBGIS, NACMAP, NACGIS, NACDraw,
FACTWORLD, NACNav, NACView, NACDatum, NAC AVIEditor,
NAC Screen Capture, 2DPoint, 3DPoint, 2DArea, etc. and many many digital maps.

NACGIS is a powerful mapping tool which can create, edit and view any kinds of maps, generates Universal Geographic Identity (UGID) for every geographic object in the world from the size 0.8x16 meters to continents and links a word document to the geographic object. This is a perfect tool for people to manage forests, real estate
properties, public works, telecommunication networks, etc..

NACDraw is a powerful tool for generating all kinds of graphic objects: polylines with different line types, width and colors, polygons with different patterns and colors, texts with different colors, font face and sizes, bitmaps and pictures from its art gallery. All its graphic objects can be either transparent or not.
Every graphic object can be linked to any other document:
a word document, sound file, NACDraw file, a file on Internet,...

NACDraw allows users to create multimedia animations which
consist of vector NACDraw graphics and sound files with various links. Users can play forward and backward, can turn frame by frame and can directly stop to edit any frame. It is a perfect tool to prepare multimedia presentations and also a powerful tool to create multimedia educational games.

FACTWORD is a world information system which collects the
latest information about every country in the world as well as
the entire world. It consists of 274 hypertext documents, 17
interactive maps and 54 statistical data sorted lists of countries
and areas in the world. Users can use a hypertext document
to get detail information about every country, use an interactive
map to retrieve a hypertext document or find a country on the
map by its name or any character string in its hypertext
document, and use a statistical data sorted list to know the
best and the worst of the world and get the relative meaning
of each actual value. Key words: GIS, UTM, MAP, NAC, Geomagnetc, Datum, WGS-84, NAD27, NAD83, MAP Server, MAPPING, AVI, Animation, Text-To-Speech, Speech Recognition, CAD Software, Navigation, GPS, GPS Software, GIS Software.

Product Name:
WEBGIS
WEBGIS makes you add intelligent maps on your web pages with

declination, inclination and intensity adds great convenience to
the maps it displays for various users.

Technology profile

Our products have incorporated the universal geocoding and grid
system - the Natural Area Coding System which assigns simple
universal geocodes to all areas on the earth, for example, NAC:
8CHK Q87P is 25mx30m area at the old location of our company.
The technology can be used in GIS systems, maps, emergenct
services, resources management, navigations etc.

The Java applet supports all kinds of objects: polylines,
polygons, texts, images, map bar charts, etc in any mixed
coordinate systems of screen pixels, longitude/latitude and UTM.

It also supports API functions to allow Java/VB scripts to
control the applet such as open files, repaint, draw traces,
switch on/off map layers, dynamic ranges and locations etc.

Its instant conversion of coordinates: pixels,
longitude/latitude, UTM and NAC and computation of geomagnetic
declination, inclination and intensity adds great convenience to
the maps it displays for various users.

WEBGIS is an ideal solution for publish various GIS projects on
web pages. More information available at:

http://www.globalserve.net/~nac/WEBGIS/
