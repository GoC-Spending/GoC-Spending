Product / Service / Licensing

Product Name:
DynacBudget
DynacBudget: Our core module that provides fully integrated Income, Balance Sheet and Cash Flow capabilities for single entities / combined entities including not only historical actual, but current reporting as well as budgets, forecasts and rolling Actual / Forecast. Full what if scenario capability all in one database driven financially intelligent application.
Built in ODBC Linkages to G/L Applications: A number of pre-built ODBC linkages are available at no cost. Alternatively, all G/L applications can be imported using CSV formatted Trial Balances or through Custom Built Linkages.
DynacCommand: A control centre which will automate file management for all modules that facilitates / schedules a number of inter-module tasks.


Product Name:
DynacViews
Financial Reporting module that out performs most G/L application built in report writers as well as 3rd party applications such as F9 and FRx Reporter.

Product Name:
DynacForecast
DynacForecast for Accounts: Our multi-user module that allows for selected G/L account secured access by managers allowing for the masses to participate in the budgeting / forecasting process complete with annotation and transaction log  allowing SOX and Bill 198 compliance.
DynacForecast SKU: Our module that allows your sales force to budget / forecast at the SKU / Customer level where selling prices and cost of goods sold data is derived directly from an ERP application or any centralized database. With the addition of SKU to Excel the distribution and collection of information can be completely automated. This application can drive not only DynacBudget but can be used to drive a Master Planning schedule(s) in various ways.


Product Name:
DynacConsolidator
DynacConsolidator: Our Consolidator module, with built in AccountManager, provides for full consolidation of any number of DynacBudget entities / combination of entities complete with built-in Cumulative Translation Adjustment (supporting Temporal Method for both self-sustaining and integrated entities) and Elimination Journal Entries. A user defined Consolidation Chart of Accounts allows mapping at any detail or summary level.
