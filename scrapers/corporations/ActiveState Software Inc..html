Company Profile


ActiveState is the leading provider of open source based
programming products and services for cross-platform
development. ActiveState's key technologies are Perl, the
Internet's most popular programming language; Python and Tcl,
user-friendly scripting languages; PHP, the dynamic Web
programming language; and XSLT, the XML transformation language.

The ActiveState Programmer Network (ASPN) offers these
technologies with the latest information and productivity tools,
empowering programmers with the freedom to work with their
preferred language and development environment.

Product / Service / Licensing

Product Name:
ActiveState PerlEx

PerlEx allows you to use Perl more efficiently on a Windows-
based web server by lowering web page delivery times and
decreasing cpu load. PerlEx 2.0 features support for Web
services.






Product Name:
ActivePerl

Free binary distribution of Perl for Linux, Solaris, and
Windows.




Product Name:
ActivePython

Free binary distribution of Python for Linux, Solaris, and
Windows.




Product Name:
ActiveTcl

Free binary distribution of Tcl for Linux, Solaris and Windows.



Product Name:
PerlMx Enterprise

PerlMx Enterprise is a customizable mail solution that provides
a complete and cost-effective means of managing mail filtering
requirements. PerlMx Enterprise, based on the PerlMx engine, is
designed to work under sendmail and includes the PerlMx engine,
ongoing updates, installation support, and a pre-determined
number of technical support hours per year. Optional filters are
available on an individual basis, and include two hours of
customization and support per filter.










Product Name:
Perl Dev Kit

The Perl Dev Kit is a suite of technologies for Windows-based
Perl programmers, and now includes PerlApp for Linux, Solaris
and Windows; and PerlMSI, for building installation packages
using Microsoft's MSI utilities. Perl programmers can implement
standalone ActiveX components, Windows services, and self-
contained applications - all written in Perl. The Perl Dev Kit
is available as a standalone package or through ASPN Perl.









Product Name:
Visual Perl

A plug-in for Visual Studio.NET, Visual Perl is a full-featured
integrated development environment that includes all the
features of a modern IDE, including a rich syntax-aware editor,
method tips, online help, and graphical debugger support.








Product Name:
Visual Python

A plug-in for Visual Studio.NET, Visual Python extends the power
of Python to the familiar Visual Studio environment and supports
a rich syntax-aware editor, method tips, online help, debugger
support, and additional Python-specific features.








Product Name:
Visual XSLT

A plug-in for Visual Studio.NET, Visual XSLT extends the
transformational power of XSLT to the familiar Visual Studio
environment and supports a rich syntax-aware editor, statement
completion, online help, debugger support and additional XSLT-
specific features, including an XPath builder.









Service Name:
ASPN

A premier service for programmers providing priority access to
the latest technologies needed to build applications with open
source tools. ASPN includes a growing archive of practical
information that will facilitate application development and
provide comprehensive solutions for increased productivity. ASPN
has three levels: ASPN Open, ASPN Komodo, and ASPN Perl.










Service Name:
ASPN Open

ASPN Open is a free online resource where programmers can freely
access ActiveState's quality-assured language distributions,
evaluation copies of ActiveState tools and components, and
essential programming information. For those learning to
program, ASPN Open provides a non-commercial licence for
ActiveState's Komodo IDE.












Service Name:
ASPN Komodo

ASPN Komodo combines a commercial license for the Komodo IDE
with free access to ActiveStates quality-assured language
distributions and essential programming information.





Service Name:
PerlDirect and PerlDirect Live

PerlDirect, and the premium PerlDirect Live, are essential
insurance policies for corporations that rely on Perl for high-
usage applications. PerlDirect is a subscription service that
delivers a quality-assured, customizable distribution of the
latest Perl and Perl tools, a comprehensive technical support
and help system, distilled Perl newsgroup information, and
direct input into future ActiveState developments. PerlDirect
Live offers all the benefits of PerlDirect, as well as a a
dedicated technical support team, professional services, and
priority response times.
















Service Name:
PythonDirect and PythonDirect Live

PythonDirect, and the premium PythonDirect Live, are essential
insurance policies for corporations that rely on Python for high-
usage applications. PythonDirect is a subscription service that
delivers a quality-assured, customizable distribution of the
latest Python and Python tools, a comprehensive technical
support and help system, and direct input into future
ActiveState developments. PythonDirect Live offers all the
benefits of PythonDirect, as well as a dedicated technical
support team, professional services, and priority response times.













Service Name:
TclDirect

TclDirect is the most reliable, comprehensive support package
for organizations that rely on Tcl for high-usage applications.
A subscription service, TclDirect delivers a quality-assured,
customizable distribution of the latest Tcl and Tcl tools, a
comprehensive technical support and help system, and direct
input into future ActiveState developments.








Service Name:
ASPN Perl

ASPN Perl combines the most advanced Perl programming technology
and information. This level adds practical Perl knowledge,
including definitive Perl books from O'Reilly, searchable and
retrievable online. In addition to the Komodo IDE, ASPN Perl
includes the Perl Dev Kit, a suite of tools to facilitate
application development; and Visual Perl, our IDE for
integrating Perl into the Visual Studio.NET environment. This
plug-in is ideal for developers familiar with Visual Studio who
want rapid Perl application development.
