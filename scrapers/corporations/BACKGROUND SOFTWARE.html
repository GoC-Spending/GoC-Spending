Company Profile


Who we are
=================================
BackGrounD Software is a privately owned corporation. There
are no hidden boards, executive figures, layers of assistants,
managers managing management or any of the faceless corporate
stuff. Pick up the phone and talk to the owner, get the decision
while you are on the phone, close the deal with a shake of the
hand.
Isn't this what business used to be ?

What we do
=================================
We work with medium to large (enterprise-level) organisations
who require independent, unbiased and reliable IT partner that
is guaranteed to be focused on the most appropriate
technological solution for their business needs. We will either
integrate or build the solution that suits the customer, from
planning and infrastructure all the way to the implementation
and maintenance.

=================================

- Incorporated in Ontario in 1997 (BGD Software Inc.)
- Incorporated in Illinois in 1998 (BGD of America, Inc.)

Unique applications:

Infrastructure Architecture and Hardware
=============================
- Infrastructure architecture assessment, design and implementation for high availability, scalability, performance and security
- Application Readiness Service
- Availability Assessment/Operational Capabilities Assessment
- Server consolidation feasibility study
- Application sizing/suitability assessment
- Data centre assessment and or design & build
- System administration (Solaris, AIX, Linux, Windows, OS390)

Infrastructure Software
=============================
- System Monitoring & Management (IBM, HP, BMC, TeamQuest)
- File and Volume Management (Symantec, HDS, Oracle, IBM, HP)
- Backup & Restore (Symantec)
- Database administration (Oracle, IBM DB2, Mysql, SQL Server)
- Application server administration (Weblogic, WebSphere)
- Business Intelligence - OLAP administration (Oracle/Hyperion, Cognos, SAS)

Installation Services
=============================
- New hardware and software installation and basic configuration (Oracle/Sun, HP, IBM, Symantec)
- Hardware and software stack upgraded and refreshes
- Equipment and data migrations and moves (local and global)
- All installation services are fixed price and fixed duration services. Please Contact us with your Hardware and Software installation requirements email us for specific hardware or software installation cost.

Configuration and Tuning Services
=============================
- We perform Configuration & Tuning services on Solaris, AIX and Linux servers.
- Server Configuration & Tuning for Application Performance
- Server Configuration & Tuning for Network Performance (TCP/IP tuning)
- Server Configuration & Tuning for Security (Server Hardening)

Storage Services
=============================
- We perform Configuration & Tuning services on HDS, IBM and EMC platforms, with Brocade fabrics.
- Storage Area Network (SAN) Configuration and/or Re-configuration for performance and availability
- Directly Attached (DA) or Network Attached Storage (NAS) Configuration and/or Re-configuration for performance and availability

Security Services
=============================
- Security Assessment
- Security Breach Damage Assessment
- Computer Forensic Service
- Security Policy Development
- Server Hardening for Solaris, Linux and AIX
- Firewall implementation

Service Management
=============================
- Creation or Assessment/Review of Support Level Agreements (SLA)
- Project or Proposal Cost/Benefit Analysis and other aspects of RFP/RFI/RFS vendor interaction
- All aspects of ITIL Service Strategy, Design, Transition and Operation

Non-managed Services - Subject Matter Expertise On Demand
=============================
We can provide a Subject Matter Expert (SME) in above mentioned categories, on a Time&Materials (T&M) basis. We will provide the most qualified individual for the tasks you require. You need to provide project management. Due to the structure of the engagement, we can not guarantee any project deliverables. It will be up to you to manage the SME's and oversee the project timelines, budget and deliverables.
