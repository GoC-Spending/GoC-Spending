Company Profile


IntelliView Technologies Inc. (IVT), a Calgary-based Canadian company, develops video analytics technologies and provides intelligent automated video surveillance systems for real-time man less detection and alarming. The solutions can be applied to a broad range of indoor and outdoor industrial applications, including remote monitoring and detection of liquid leaks (e.g. oil, waste water, bitumen, acid, chemicals, and other fluid products), and wet gas leaks (e.g. ethane, propane, butane, and other natural gas liquids), flare stack flame,perimeter security threats (intrusion, theft, fence crossing), wildlife intrusion, and critical heating/cooling (temperature sensitive processes).

Video analytics and patented image processing technologies are applied to live video feed. High detection accuracy and low false alarms performance - from glare, snow, heavy rain, shadowing and false objects- are achieved through the application of advanced environmental filters, video analyzing software, and object definition tools. Solutions are optimized based on customer needs by an in-house development team.

With timely access to critical field information and instant image/video event evidence, decision makers and responders are able to address situations quickly and efficiently. Early detection and action help to reduce or prevent fines, clean-up costs, product loss, property damage, and other negative consequences of delayed access to field intelligence.

Platforms: Dual Camera Analytic Module or DCAM™ (a compact, integrated, wireless system), fixed system of individualized components (Generation 1), self-contained trailer (fuel cell / solar powered), and C1D1.

Product / Service / Licensing

Product Name:
IntelliView Pipeline Leak (Liquid and Wet Gas) Detection and Automated Alerting System
The IVT Liquid Pipeline Leak Detection System consists of IntelliView’s advanced proprietary leak analytics, high quality thermal/color cameras, and a hybrid video recorder (HVR) with image processing and alarming software, which can be integrated into SCADA and other networks. In any lighting and most weather conditions, the system detects fluid sprays, pooling and spills as well as wet gas leaks (butane, propane, NGL, etc.) as little as 0.36 m³/hour in real time. Detected liquid products include oil, bitumen, brine, waste water, acid, and chemicals. Within seconds, it generates an alarm notification (via web access user interface and email) with image and video evidence for instant event verification based on user-defined conditions. The solution is designed for above-ground monitoring in industrial settings, including oil and gas, production/processing plants, mining, and water facilities. Click for more: http://intelliviewtech.com/solutions/industrial-fluid-leak/.

Product Name:
IntelliView Flare Stack Monitoring and Automated Alerting System
The IVT Flare Stack Monitoring System consists of high quality thermal/color cameras, and a hybrid video recorder (HVR) with image processing and alarming software. It provides a wide thermal temperature sensing range of -40°C to 550°C with accuracy to 2° C and accurately detects the absence or ignition of flame (as defined by the user), in any lighting and most weather conditions. Within seconds, it generates an alarm notification (via web access user interface and email) with image and video evidence for instant event verification. The solution can be integrated onto existing and third-party monitoring systems. Click for more: http://intelliviewtech.com/solutions/continuous-flare-stack-flame-monitoring/.

Product Name:
IntelliView Asset Monitoring and Automated Alerting System
The Asset Monitoring System consists of IntelliView’s suite of security analytics, thermal/color cameras, and hybrid video recorder (HVR) with image processing and alarming software. In any lighting and most weather conditions, the system detects perimeter intrusion, loitering, one-way/two-way fence/line crossing, object taken, object left and other security related events in real time. Within seconds, it generates an alarm notification (via web access user interface and email) with image and video evidence for instant event verification based on user-defined conditions. The solution can be integrated onto existing and third-party security devices, including audible and visual alarms, and access control systems located onsite. Click for more: http://intelliviewtech.com/solutions/asset-monitoring/.

Product Name:
IntelliView Wildlife Detection and Automated Alerting System
The Wildlife Detection System consists of IntelliView’s proprietary wildlife analytics, thermal/color cameras, and hybrid video recorder (HVR) with image processing software. In any lighting and most weather conditions, the system detects animals (e.g. birds) entering a perimeter, such as sensitive environments and dangerous areas (toxic tail ponds), and generates an alarm notification (web access user interface and email) with image and video evidence for instant event verification. Within seconds, it generates an alarm notification (via web access user interface and email) with image and video evidence for instant event verification based on user-defined conditions. The solution can be integrated with deterrents (air cannons, sirens, lights) for remote and automated control. Click for more: http://intelliviewtech.com/solutions/wildlife-monitoring-detection/.

Product Name:
IntelliView Dual Camera Analytic Module (DCAM™)
(New Product, Released in Q1 of 2016) A second generation technology provided as a hardware component of an IntelliView solution, the DCAM™ is an integrated intelligent camera system designed for monitoring and detection of industrial liquid pipeline leaks and perimeter intrusion. Featuring a weatherproof, compact design, the unit houses the following components: a high resolution thermal imager (has twice the thermal senility of generation 1 thermal camera), a high definition optical sensor, image processor with video analytics, data storage, and climate controlled heater and cooler. Alarm notifications with event image and video, which are triggered by user-defined events, are transmitted wirelessly or via Ethernet connectivity. Live feeds, alarms notifications, alarm videos and settings are accessible 24/7 from internet-enabled devices via a web interface. DCAM™ accessories include solar panels and a GPS.
