Product / Service / Licensing

Product Name:
Aircraft Maintenance Manager
The possibilities of Aircraft Maintenance Manager are close to infinite. Users can monitor everything from aircraft maintenance tracking, employees' recurrent training, licensing and medical renewals, heavy equipment maintenance tracking and much more! It is also possible to include all the electronic manuals such as the aircraft maintenance manuals, the pilot training manuals, etc. All this information can be made available to authorized users from anywhere in the world via the Internet.

Product Name:
Inventory Manager
Our Inventory Manager software is the perfect tool to manage purchases, invoices, cost statistics, etc. With a single click, the user controlling inventory is able to transfer an item from the inventory shelf to an aircraft work order. With another click, the removed item automatically goes to quarantine if it is a rotable item. If it is a consumable, an entry for scrap is created. From there, a packing slip can automatically be generated to send the item for repair or overhaul, or it can be returned to the inventory with its current time for trouble shooting purpose. Inventory Manager tracks time on components and items to be transferred to and/or from an aircraft. The tracking time will follow. For each part movement, a complete history record is created.

A tracking number is attributed to each part entering the system. This tracking number will allow the user to enter essential information concerning the part, such as the batch or serial number, the suppliers name, the expiration date, etc. To identify the parts, our system can create a label filled with the required information and a bar code. All part movements are registered in an easy to consult history record. With a simple click, it is very easy to find out where a part was installed, to whom and when it was sold or from which supplier it was purchased.

Product Name:
User Module
Security and users management are very important in computerized aircraft maintenance, this is why we developed that great utility that prevents non-authorized persons from accessing the program. Account Manager lets the system administrator create personalized accounts for every person who needs to access the programs. You can easily restrict or grant access to each user for different program features by selecting them from a list, thus reducing the risks of mistakes.

Product Name:
Transport Canada AD's Tracker
As you already know, AD's follow-up is one of the most important tasks in aircraft maintenance. We designed this tool to help you verify and keep track of all AD's that apply to your aircraft. Using the Internet, this program will fetch new AD's from the Transport Canada website and display the AD's for each of your aircraft. You will then be able, with a single mouse click, to read it and see if it applies to that specific aircraft or not.



Product Name:
Air Keeper
The Air Keeper module was developed from a customer's request. This person employs a dispatch person to assign the flights to the aircraft of the fleet. This module was designed to alert that person when an aircraft enters a state of "warning" for a maintenance task and that, without having to consult the Aircraft Maintenance Manager program. The "warnings" used in this program are created to reflect the operations and not for maintenance planning as in Aircraft Maintenance Manager.

Here is an example: If an update of the fleet aircraft time is made at a maximum interval of 10 hours, 3 days or 10 cycles, the software "warnings" must be adjusted with this data. Therefore, when the time remaining before the achievement of a maintenance task reaches these limits, the registration number of that aircraft will appear on the screen with the remaining hours in order to warn the supervisor.

This tool is essential to guarantee that the execution of tasks will be done within prescribed times.

Product Name:
Component Work Report
This module creates the work orders that cover the component overhaul or repair. It also allows a direct link with Inventory Manager for easy invoicing or internal cost control. This is the ideal device for shops conducting overhauls and/or repairs on any component types.

Product Name:
AMM Remote Reporting
The AMM Remote Reporting Module (Web-based module) allows the consultation of Aircraft Maintenance reports and the consultation of Inventory from anywhere in the world via the Internet. Pilots can also update aircraft time on the internet and even on their cell phone!

Product Name:
Electronic Signature
This module eliminates the need to keep paper copies of the maintenance certification documents for signature purposes. Signatures and initials are electronically added and a certification log is created for each employee.

Product Name:
Time and Employee Management
The Time Manager Module is a great system to track the time of the employees on a specific task. It helps to know how many hours an employee has spent on a task and to track the clock in and clock out.
