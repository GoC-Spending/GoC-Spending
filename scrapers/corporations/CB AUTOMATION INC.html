Company Profile


CB Automation Inc. is a full service software supplier to the
Industrial sector in Eastern Canada. We supply packaged
automation software and Instrumentation solutions for HMI, SCADA, Batch, data
Acquisition, process control and reporting systems including factory
floor automation, downtime reporting, quality control, work in
progress, lot tracking, genealogy tracking, validation, plant
wide historian and networking.

Service Name:
Automated Analytical System
Automated Analytical System: Specialty Chemical Plant -
Environmental regulation compliance


The application is set up to control the collection of effluent
samples directly to the analytical equipment in real time. By
controlling valves and pumps, samples can be collected
automatically, and the results from the analyzers are stored on
hard disk for archival purposes. Reports are also sent by fax
automatically to supervisors. This reduces the testing time by
about 60% and provides real time effluent information to the
people that need it

Success stories:

Productivity increase: Automotive: Machine control
In the case of this Ontario based automotive
company the question was how could they increase
production of vehicles without major capital
injection. Although they had in the past
identified problem areas, all effort in acquiring
and analyzing the data was done manually. This
took several man-hours and the reports were
always disputed for accuracy.


Initial attempts to automate the collection of
data with Microsoft's MS Access, enhanced with
the existing SCADA system and custom code proved
unsuccessful. MS Access was not designed to work
in Real-Time applications and custom code proved
too costly to maintain.


CB Automation Inc. offered an off the shelf and
Scalable solution using packaged real time
software from Intellution. Products used for the
project were iDowntime, iFix and Intellution's
new high-speed historian package. Two iFix
Servers were installed to talk to fifty (50)
brand name PLC's. An archiving server with high-
speed historian and iDowntime was installed on an
independent PC connected to the two iFix servers
via Ethernet. MS Excel add-in was also utilized
for its off the shelf WEB ready reporting
capability.


With the off the shelf solution, Management,
Engineers and Production supervisors are all able
to access validatable uptime and downtime
information on every machine unit in their
manufacturing process. The key aspect for the
Managers was to view reports in MS Excel, which
indicated Production vs. parts in the buffers and
cycle time of machine vs. production . The
Engineers were able to analyze downtime by areas
and continually enhance cycle time. This downtime
reduction and cycle time improvement is the key
to getting higher production numbers from
existing equipment. With the use of High-speed
Historian they are able to archive every single
event with sub second accuracy. Then go back in
time and remodel their criteria for analyzing.
Plant Supervisors were able to view production
numbers and their staff were able enter downtime
reasons directly on the system with a simple drop
down menu saving time and getting accurate time
stamped downtime entries.


In addition to all the benefits of a real-time
system, they were able to save the man-hours that
had been spent entering the information manually.
Within a few days of the system up and running ,
they were able to identify re-occurring downtime
problems by area and are showing a 4% increase in
their production with a very small investment.
Based on the success of this project a second
department has began a similar project two other
departments are preparing to start using our
solution.
